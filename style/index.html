Laboratorium 1: git

Przykładowe użycie standardowych znaczników html

Praca z git:

  • Wykonaj i zapisz zmiany w kodzie projektu
  • Przejdz do panelu "Source Control"
  • Ustaw zmodyfikowanym plikom stan "Staged"
  • Wprowadz opis do pola tekstowego Messege
  • Zatwierdz zmiany lokalne (commit)
  • Pobierz i scal zmiany zdalne(pull)
  • Rozwiąż ewentualne problemy
  • Wyślij aktualną wersję na serwer (push)
